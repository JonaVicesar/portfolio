## Getting Started

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/JonaVicesar/portfolio.git
   ```

2. Navigate to the project directory
   ```bash
   cd portfolio
   ```

3. Open `index.html` in your browser or use a live server
